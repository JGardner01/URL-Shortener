About URL Shortener

URL Shortener helps you to shorten long URLs into shorter, more manageable links, making it easier to share and remember links.

This URL Shortener application was created by James Gardner.
Relevant Links: Github Repo | Portfolio

Features

  • Dashboard: Your URLs can be viewed, edited, deleted, shared and monitored within the dashboard.
  • User Accounts: Continue to monitor and save your shortened URLs with an account.
  • Customised Short Codes: Shortened links can be customised to be easily memorable.
  • QR Code Generation: QR codes are automatically generated for each shortened link.
  • Password Protection: Secure your links by adding a password to control who can access the link.
  • Expiration Dates: A custom expiration date and time can be set, after which the shortened URL will no longer be accessible. Default expiration date is 30 days.
  • Click Tracking: Track how many times your shortened URL has been clicked to monitor usage.
  • Click Limits: Limit how many times a shortened URL can be accessed.
  • Google Safe Browsing Integration: All URLs are checked against Google's Safe Browsing database to ensure that shortened links are safe to access.

How to Use

  1. Enter the long URL that you would like to be shortened.
  2. Optionally, customise the new URL with a custom short code or add options such as password protection, a custom expiration date, or click limit.
  3. Click "Submit" to generate a completely, unique shortened link and QR code.
  4. Share and use the provided short URL or QR code.
  5. Manage and track your shortened URLs in the dashboard.
  6. Save and continue monitoring your shortened URLs by creating/logging into an account.
